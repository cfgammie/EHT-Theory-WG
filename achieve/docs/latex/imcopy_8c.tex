\subsection{Detailed Description}
Copy the input F\+I\+TS image to the new output file with optional data compression. 

Use\+: imcopy inimage\mbox{[}ext\mbox{]} outimage\mbox{[}compress\mbox{]}

Copy the input F\+I\+TS image to the new output file with optional data compression. The output image maybe externally compressed with the gzip or Unix compress algorithm, or it may be internally compressed with the new tile compression format. The output image may be written in the tile-\/compressed F\+I\+TS image format by adding the compression qualifiers in square brackets following the output file name. Examples\+:


\begin{DoxyItemize}
\item imcopy infile.\+fit \textquotesingle{}outfile.\+fit\mbox{[}compress\mbox{]}\textquotesingle{}
\end{DoxyItemize}

This will use the default compression algorithm (Rice) and the default tile size (row by row)


\begin{DoxyItemize}
\item imcopy infile.\+fit \textquotesingle{}outfile.\+fit\mbox{[}compress G\+Z\+IP\mbox{]}\textquotesingle{}
\end{DoxyItemize}

This will use the G\+Z\+IP compression algorithm and the default tile size (row by row). The allowed compression algorithms are Rice, G\+Z\+IP, and P\+L\+IO. Only the first letter of the algorithm name needs to be specified.


\begin{DoxyItemize}
\item imcopy infile.\+fit \textquotesingle{}outfile.\+fit\mbox{[}compress G 100,100\mbox{]}\textquotesingle{}
\end{DoxyItemize}

This will use the G\+Z\+IP compression algorithm and 100 X 100 pixel tiles.


\begin{DoxyItemize}
\item imcopy infile.\+fit \textquotesingle{}outfile.\+fit\mbox{[}compress R 100,100; 4\mbox{]}\textquotesingle{}
\end{DoxyItemize}

This will use the Rice compression algorithm, 100 X 100 pixel tiles, and noise\+\_\+bits = 4 (assuming the input image has a floating point data type). Decreasing the value of noisebits will improve the overall compression efficiency at the expense of losing more information.


\begin{DoxyItemize}
\item imcopy infile.\+fit outfile.\+fit
\end{DoxyItemize}

If the input file is in tile-\/compressed format, then it will be uncompressed to the output file. Otherwise, it simply copies the input image to the output image.


\begin{DoxyItemize}
\item imcopy \textquotesingle{}infile.\+fit\mbox{[}1001\+:1500,2001\+:2500\mbox{]}\textquotesingle{} outfile.
\end{DoxyItemize}

This extracts a 500 X 500 pixel section of the much larger input image (which may be in tile-\/compressed format). The output is a normal uncompressed F\+I\+TS image.


\begin{DoxyItemize}
\item imcopy \textquotesingle{}infile.\+fit\mbox{[}1001\+:1500,2001\+:2500\mbox{]}\textquotesingle{} outfile.\+fit.\+gz
\end{DoxyItemize}

Same as above, except the output file is externally compressed using the gzip algorithm.
