Generates a grid of coordinates along the image plane. 

It generates an equidistant grid of Na alpha-\/points and of Nb beta-\/points and stores the two grids in the double arrays aco\mbox{[}\mbox{]} and bco\mbox{[}\mbox{]}. The grid are from -\/amax -\/$>$ amax and from -\/bmax -\/$>$bmax. Both Na and Nb need to be odd. The routine checks whether the number of grid points is less than the maximum numbers N\+A\+M\+AX and N\+B\+M\+AX stored in the global definitions.

Converts a polar k-\/m map to an image. 

Given a grid of (Nk,Nm) points of the polar transform (with the real part stored in Re\+Pk\mbox{[}\mbox{]}\mbox{[}\mbox{]} and the imaginary part in Im\+Pk\mbox{[}\mbox{]}\mbox{[}\mbox{]}), it returns the corresponding image on a (Na,Nb) grid of points, with the grid of alpha and beta points stored in aco\mbox{[}\mbox{]} and bco\mbox{[}\mbox{]} and the image stored in Image\mbox{[}\mbox{]}\mbox{[}\mbox{]}.

It will only use the k-\/modes between kmin and kmax as well as the m-\/modes between mmin and mmax, as given in the inputs

It uses the trapezoid rule to perform the integral over k and a simple summation for m. NB\+: it assumes that the k-\/coordinates are equidistant.
