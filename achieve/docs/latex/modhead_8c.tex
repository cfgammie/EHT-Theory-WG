\subsection{Detailed Description}
Writes or Modifies the value of a header keyword. 

Use\+: modhead infile\mbox{[}ext\mbox{]} keyword newvalue

This program can write or modify the value of a header keyword. If the 4th \textquotesingle{}newvalue\textquotesingle{} parameter is not specified then the program just prints out the current value of the keyword. Otherwise, it replaces the value of the keyword if it exists, or writes a new keyword if it doesn\textquotesingle{}t exist.

Examples\+:


\begin{DoxyItemize}
\item modhead file.\+fits dec -\/ list the value of the D\+EC keyword
\item modhead file.\+fits dec 30.\+0 -\/ write or modify the D\+EC keyword
\end{DoxyItemize}

This program opens the input file and attempts to read the specified keyword. If the keyword exists, the current value is printed out. If the 4th \textquotesingle{}newvalue\textquotesingle{} argument is specified then the program first checks to make sure that we are not attempting to modify a structural keyword (like N\+A\+X\+IS) which could corrupt the file format. It then constructs a new keyword record string with the new value and overwrites or appends that record to the header.
