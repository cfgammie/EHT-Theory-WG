\subsection{Detailed Description}
Converts an image to u-\/v maps (complex amplitude and phase) 

This program reads an image stored in an input F\+I\+TS file, calculates its complex Fourier transform, and stores the resulting visibility amplitudes and phases in an output O\+I\+F\+I\+TS file.

Use\+: image2uv \mbox{[}-\/sv\mbox{]} \textquotesingle{}infile.\+fits\textquotesingle{} \mbox{[}-\/o \textquotesingle{}uvresults.\+out\textquotesingle{}\mbox{]}

The required options are\+:
\begin{DoxyItemize}
\item \char`\"{}filename\char`\"{}\+: sets the input image filename (F\+I\+TS)
\end{DoxyItemize}

The optional options are\+:
\begin{DoxyItemize}
\item \char`\"{}-\/o \mbox{[}filename\mbox{]}\char`\"{}\+: sets the output visibility filename (U\+V\+F\+I\+TS).
\item \char`\"{}-\/s\char`\"{}\+: silent mode. It does not print anything
\item \char`\"{}-\/v\char`\"{}\+: verbose mode. It prints a lot more information
\end{DoxyItemize}

If no options are given, it prints a help message

Examples\+:


\begin{DoxyItemize}
\item image2uv inimage.\+fits -\/o uvresults.\+out
\end{DoxyItemize}

Reads the file form inimage.\+fits and outputs the visibility amplitudes and phases into uvresults.\+out
