Reads visibilities on the u-\/v plane from the external file \char`\"{}filename\char`\"{}. 

The file is assumed to have 4 A\+S\+C\+II columns, separated by white spaces\+: U coordinate, V coordinate, Visibility Amplitude, Visibility Phase. The rows are assumed to be organized in such a way that for each value of the U coordinate, all V coordinates are listed sequentially. The grid is assumed to be equidistant along both directions, and this is checked during the reading. On return all columns are stored in the relevant arrays in the argument list.
