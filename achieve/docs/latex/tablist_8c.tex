\subsection{Detailed Description}
Lists the values in all the rows and columns of the input table. 

Use\+: tablist infile\mbox{[}ext\mbox{]}\mbox{[}col filter\mbox{]}\mbox{[}rowfilter\mbox{]}

Lists the values in all the rows and columns of the input table. In order to reduce the amount of output, it is sometimes useful to select only certain rows or certain columns in the table to be printed by filtering the input file as shown in the examples.

Examples\+:
\begin{DoxyItemize}
\item tablist \textquotesingle{}intab.\+fit\mbox{[}E\+V\+E\+N\+TS\mbox{]}\textquotesingle{} -\/ print \textquotesingle{}E\+V\+E\+N\+TS\textquotesingle{} table
\item tablist \textquotesingle{}intab.\+fit\mbox{[}1\mbox{]}\textquotesingle{} -\/ print 1st extension
\item tablist \textquotesingle{}intab.\+fit\mbox{[}1\mbox{]}\mbox{[}col X;Y\mbox{]}\textquotesingle{} -\/ print X and Y columns
\item tablist \textquotesingle{}intab.\+fit\mbox{[}1\mbox{]}\mbox{[}P\+HA $>$ 24\mbox{]}\textquotesingle{} -\/ print rows with P\+HA $>$ 24
\item tablist \textquotesingle{}intab.\+fit\mbox{[}1\mbox{]}\mbox{[}col X;Y\mbox{]}\mbox{[}P\+HA $>$ 24\mbox{]}\textquotesingle{} -\/ as above
\item tablist \textquotesingle{}intab.\+fit\mbox{[}1\mbox{]}\mbox{[}col X;Y;Rad=sqrt(X$\ast$$\ast$2+\+Y$\ast$$\ast$2)\mbox{]} -\/ print X, Y, and a new \textquotesingle{}Rad\textquotesingle{} column that is computed on the fly.
\end{DoxyItemize}

This program opens the file (which may have been filtered to select only certain rows or columns in the table) and checks that the H\+DU is a table. It then calculates the number of columns that can be printed on an 80-\/column output line. The program reads and prints out the values in these columns row by row. Once all the rows have been printed out, it repeats the process for the next set of columns, if any, until the entire table has been printed. All the column values are read as formatted character strings, regardless of the intrinsic datatype of the column. C\+F\+I\+T\+S\+IO uses the T\+D\+I\+S\+Pn keyword if it exists to format the string (e.\+g., T\+D\+I\+S\+P1 = \textquotesingle{}F8.\+2\textquotesingle{}, or T\+D\+I\+S\+P2 = \textquotesingle{}E15.\+6\textquotesingle{}), otherwise it uses a default display format for each column datatype. It should be cautioned that reading a table this way, one element at a time column by column and row by row, is not very efficient. For large tables, it is more efficient to read each column over least 100 rows at a time into temporary arrays. After processing all these rows, then read the next 100 rows from the table, and so on, until all the rows have been processed.
